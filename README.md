### Introduction
This is a tutorial on how to implement a simple program
to solve the Hartree-Fock with SCF iterations.

### Dependencies
- `numpy`
- `scipy`
- If you wish to use `./data/gen_data.py` to generate the integrals, `pyscf` is also required.
